## Run application

1 - Fork this repository

2 - Install dependencies:

```shell
npm install
```

or

```shell
yarn
```

3 - Create a `db.json` file in the application root with the contents below:

```json
{
  "tasks": []
}
```

4 - Run server:

```shell
npm run jsonserver
```

or

```shell
yarn jsonserver
```

5 - Run application:

**iOS**

```shell
npm run ios
```

or

```shell
yarn ios
```

**Android**

```shell
npm run android
```

or

```shell
yarn android
```
